productionprofiles.profiledata.ProfileFile#

class productionprofiles.profiledata.ProfileFile(fc: dict[slice(<class 'str'>, <class 'pandas.core.frame.DataFrame'>, None)])#

Bases: object

Object representing both maps and well data content. Maps represents relationships between reservoirs, wells, drill centers and flow loops. Wells can be grouped using different methods:

  • wells listed by name w/o filtering or based on service type (production, injection).

  • wells classified by reservoirs, drill centers, flow loops or service type.

  • reservoirs, drill centers, flow loops classified by service type.
